World Tweets: Real time mapping of tweets around the world.

What you see below are tweets hitting the internet in real-time, along with the location where they were tweeted from. Tweets are removed from the map every 10 seconds to make room for new ones.

Tweets are sampled from Twitter's Streaming API using node.js, delivered to your browser via socket.io, and mapped using D3.js. Code available at https://github.com/abailin/worldtweets.
